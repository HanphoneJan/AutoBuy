📖 使用帮助

快速开始

1

安装依赖

确保已安装 Python 3.8+，并安装项目依赖：

uv sync
2

启动应用

运行以下命令启动 Web 服务：

uv run app.py
3

访问界面

在浏览器中打开：

http://localhost:5000

京东抢购说明

  • 时间设置：格式为 YYYY-MM-DD HH:MM:SS.ffffff，例如：2024-11-01 20:00:00.000000
  • 时间校准：系统会自动获取京东服务器时间并校准本地时间差
  • 自动跳转：登录后自动跳转到订单确认页面
  • 精准提交：到达时间后自动点击提交订单按钮

淘宝抢购说明

  • 手动准备：需要手动登录并勾选购物车商品
  • 双线程：主线程监控结算按钮，高频线程快速点击提交订单
  • 实时监控：实时检测页面状态并自动操作
  • 多种点击方式：支持常规点击和 JavaScript 点击

⚠️ 注意事项

  • 请确保已安装 Chrome 浏览器
  • 首次运行可能需要下载 ChromeDriver（程序会自动处理）
  • 请勿用于违规抢购，仅限学习交流使用
  • 建议提前测试登录和页面跳转流程
  • 抢购成功后请尽快付款

常见问题

Q: 浏览器启动失败怎么办？

A: 请检查是否安装了 Chrome 浏览器，或尝试更新 ChromeDriver 版本。

Q: 登录二维码扫描后没反应？

A: 请确保网络连接正常，或尝试刷新页面重新扫描。

Q: 抢购时间已到但没有自动提交？

A: 检查时间格式是否正确，或查看日志了解具体错误信息。
